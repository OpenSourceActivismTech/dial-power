# phonebank
A phone banking solution powered by Twilio

# dev prerequisites
- node v6.11.1
- npm v3.10.10
- docker v17.06.0
- postgres v9.6.3

# `make` commands
- `make install` - install local dependencies via npm
- `make lint` - eslint the repository
- `make build` - build a Docker image
- `make run` - Run the latest Docker image
